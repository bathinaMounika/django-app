# Web Application To Conduct Online Tests

## Requirement
<ul>
    <li>
        python 3.7.1
    </li>
</ul>

### Setup
<ul>
<li>Clone the repo</li>
</ul>

```
git clone https://github.com/bathinaMounika/django-app.git
```
<ul>
    <li>
        Create Virtual Environment
    </li>
<ul>
```
python -m virtualenv DJANGO
```
```
DJANGO/Scripts/activate.bat    
```
<ul>
    <li>
        install django
```
pip install Django
```
    </li>
</ul>
